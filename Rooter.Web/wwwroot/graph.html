How to Use This Tool

🗂️ File Path

Directly specify the path to your project.assets.json file(s). Best for analyzing specific projects.

  • Single project: Enter one file path
  • Multiple projects: Enter multiple paths, one per line
  • Example: C:\MyProject\obj\project.assets.json

🔍 Auto Discover

Automatically find all project.assets.json files in a directory tree. Perfect for analyzing entire solutions.

  • Enter a directory path (or leave blank for current directory)
  • Select which projects to analyze
  • Great for solution-wide dependency analysis

📁 Finding project.assets.json

The project.assets.json file is generated after restoring NuGet packages:

  • Run dotnet restore in your project directory
  • File location: YourProject\obj\project.assets.json
  • Each project has its own file in its obj folder
